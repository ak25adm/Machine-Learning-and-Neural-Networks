# Machine-Learning-and-Neural-Networks
How Backpropagation improves Neural Network Accuracy During Training
Neural Network Training using Backpropagation

This project contains two machine learning models:

1. Car Price Prediction (Regression)
2. Diabetes Prediction (Classification)

---

Objective

The main goal of this project is to show how increasing the number of epochs (backpropagation cycles) improves the performance of neural networks.

---

What I Did

- Built neural network models using TensorFlow/Keras  
- Trained both models with different numbers of epochs  
- Observed how performance changes as epochs increase  

---

Key Idea

Neural networks learn using a process called **backpropagation**.

- In each epoch, the model updates its weights  
- More epochs = more learning  
- This leads to better predictions  

---

Results

## Car Price Prediction
- Low epochs → high error  
- More epochs → lower error (better predictions)

## Diabetes Prediction
- Low epochs → lower accuracy  
- More epochs → higher accuracy  

---

Conclusion

Increasing the number of epochs improves model performance in both cases.

This proves that:
- Learning in neural networks is gradual  
- More backpropagation leads to better results

Tools Used

Note

Very high epochs can cause overfitting, so a balance is important.
